Обзор Проекта

Название

Игра 2048 — логическая головоломка с целью создания плитки 2048

Назначение

Создание классической логической игры для развлечения и развития стратегического мышления.

Цели Проекта

  • Реализовать игровое поле и его отрисовку в консоли
  • Реализовать логику движения и объединения плиток в четырех направлениях
  • Реализовать систему добавления новой плитки после каждого хода
  • Реализовать систему подсчета очков
  • Реализовать систему окрашивания плиток
  • Добавить функцию отмены хода (Undo)
